## 🚀 使用方法

### 步骤1：启动React应用

**方式1：使用启动脚本（推荐）**

在浏览器中打开主项目：

## ⚠️ 注意事项

1. **使用前必须先启动React应用**
   ```bash
   ./start-accounting-stats.sh
   ```

2. **确保端口3001未被占用**
   ```bash
   lsof -i :3001
   ```

3. **如需修改端口**
   - 修改 `记账统计/package.json` 中的端口
   - 同步更新 `index.html` 中iframe的src

## 🐛 故障排查

### 问题：点击统计按钮显示错误页面？

**原因**：React应用未启动或端口被占用

**解决方案**：
1. 检查React应用是否已启动：
   ```bash
   lsof -i :3001
   ```

2. 如果没有启动，运行启动脚本：
   ```bash
   ./start-accounting-stats.sh
   ```

3. 如果端口被占用，修改端口配置

### 问题：npm install失败？

**解决方案**：
```bash
cd 记账统计
rm -rf node_modules package-lock.json
npm install
```

### 问题：React应用启动失败？

**解决方案**：
1. 检查Node.js版本（需要 >= 18）
   ```bash
   node -v
   ```

2. 清理缓存重新安装
   ```bash
   npm run clean
   npm install
   ```
