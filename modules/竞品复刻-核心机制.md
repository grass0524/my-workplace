## 逐帧分析机制（⭐ 核心机制）

**核心原则**：每个页面状态变化都要作为独立"帧"进行分析，确保不遗漏任何页面状态。

### 帧识别规则

以下情况视为新帧：
1. **页面URL变化**（新页面）
2. **顶部Tab切换**（每个顶部Tab内容视为新帧）
3. **内容Tab切换**（每个内容Tab视为新帧）
4. **嵌套Tab切换**（Tab下的Tab视为新帧）
5. **对话框打开**（对话框内容视为新帧）
6. **侧边栏/抽屉打开**（侧边栏内容视为新帧）
7. **筛选/搜索后内容变化**（变化后的内容视为新帧）
8. **表单展开/折叠**（展开后的内容视为新帧）
9. **排序后内容变化**（表格排序后的内容视为新帧）
10. **分页切换**（分页后的内容视为新帧）
11. **数据刷新**（自动刷新或手动刷新后的内容视为新帧）
12. **批量操作后内容变化**（批量删除、批量修改后的内容视为新帧）

### 逐帧分析流程

对每个帧执行：
1. **帧标识**：记录帧ID、帧类型、触发元素、父帧ID
2. **帧截图**：保存当前帧的截图
3. **帧元素提取**：提取当前帧的所有元素（使用区域扫描检查清单）
4. **帧交互测试**：测试当前帧的所有交互元素
5. **帧业务规则提取**：提取当前帧的业务规则
6. **子帧发现**：发现当前帧触发的子帧（Tab、对话框等）
7. **帧记录**：记录帧的所有信息到JSON文件

### 帧记录格式

```json
{
  "frameId": "frame-001",
  "frameType": "page/topTab/contentTab/nestedTab/dialog/sidebar/filter/sort/pagination/refresh/batchOperation",
  "pageUrl": "页面URL",
  "triggerElement": "触发元素",
  "parentFrameId": "父帧ID",
  "timestamp": "时间戳",
  "screenshot": "截图路径",
  "elements": {
    "topArea": [],
    "topRightArea": [],
    "leftArea": [],
    "contentArea": [],
    "bottomArea": [],
    "floatingElements": []
  },
  "tabs": {
    "topTabs": [],
    "contentTabs": [],
    "nestedTabs": []
  },
  "interactions": [],
  "businessRules": [],
  "childFrames": []
}
```

## 区域扫描检查清单（统一模板）

## Tab递归测试机制（⭐ 核心机制）

### Tab分类识别

- **顶部Tab**：页面级别的Tab（如"总览"、"运维中心"），位于页面顶部，可关闭
- **内容Tab**：内容区域的Tab（如"告警中心/通知公告"），位于内容区域
- **嵌套Tab**：Tab下的Tab（如"电站信息"Tab下的"功率/发电量"Tab）

### Tab递归测试流程

对每个Tab组件执行：
1. **Tab识别**
   - [ ] 识别Tab类型（顶部Tab/内容Tab/嵌套Tab）
   - [ ] 提取Tab标签文字
   - [ ] 记录Tab层级和父Tab ID

2. **Tab切换**
   - [ ] 点击Tab标签
   - [ ] 等待内容加载（2-3秒）
   - [ ] 保存Tab切换后的截图

3. **Tab内容分析**（作为新帧处理）
   - [ ] 重新扫描Tab内容区域的所有元素（使用区域扫描检查清单）
   - [ ] 检查是否有嵌套Tab，如有则递归执行
   - [ ] 提取Tab下的所有交互元素
   - [ ] 测试Tab下的所有交互元素

4. **Tab功能测试**
   - [ ] 测试Tab关闭按钮（如有）
   - [ ] 测试Tab切换动画/效果
   - [ ] 记录Tab切换后的页面变化

5. **Tab记录**
   - [ ] 记录Tab层级关系
   - [ ] 记录Tab内容元素
   - [ ] 记录Tab交互测试结果

### Tab测试记录格式

```json
{
  "tabId": "tab-001",
  "tabType": "topTab/contentTab/nestedTab",
  "tabText": "Tab标签文字",
  "tabLevel": 1,
  "parentTabId": null,
  "frameId": "frame-XXX",
  "tested": true,
  "contentElements": {
    "buttons": [],
    "forms": [],
    "tables": [],
    "nestedTabs": []
  },
  "screenshot": "截图路径",
  "interactions": []
}
```
