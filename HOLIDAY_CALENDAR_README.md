# 假期日历功能说明

## 功能概述
成功为放假倒计时卡片添加了假期日历功能，用户可以点击卡片查看完整的假期安排和补班信息。

## 主要特性

### 1. 放假倒计时卡片增强
- 添加了日历图标按钮
- 整个卡片可点击打开日历
- 添加了悬停效果，提供视觉反馈

### 2. 假期日历弹窗
- 显示完整的月度日历视图
- 支持月份切换（上个月/下个月）
- 标注法定假期和调休补班日期
- 点击日期查看详细信息

### 3. 假期数据（2026年）
包含以下节假日的完整信息：
- 元旦：1月1日-1月3日
- 春节：2月15日-2月21日
- 清明节：4月4日-4月6日
- 劳动节：5月1日-5月5日
- 端午节：6月19日-6月21日
- 中秋节：9月25日-9月27日
- 国庆节：10月1日-10月7日

### 4. 补班日期（2026年）
- 元旦补班：1月4日
- 春节补班：2月14日、2月22日
- 清明补班：4月12日
- 五一补班：5月9日
- 中秋补班：9月27日
- 国庆补班：10月10日

### 5. 颜色方案
- **绿色**：法定假期（使用成功色 `var(--success-color)`）
- **黄色**：调休补班（使用警告色 `var(--warning-color)`）
- 颜色设计参考健康打卡模块，保持整体风格一致
- 使用半透明背景和柔和的配色方案

## 使用方法

### 查看假期日历
1. 点击放假倒计时卡片任意位置
2. 弹窗显示当前月份的日历
3. 使用左右箭头切换月份

### 识别日期类型
- **绿色背景 + "假"标签**：法定假期
- **黄色背景 + "班"标签**：调休补班
- **普通背景**：正常工作日或周末

### 查看日期详情
1. 点击日历中的任意日期
2. 底部显示该日期的详细信息：
   - 假期：显示假期名称和日期范围
   - 补班：显示补班名称
   - 周末：显示正常休息
   - 工作日：显示正常上班

## 技术实现

### HTML 结构
- 放假倒计时卡片添加了 `onclick` 事件
- 新增 `holiday-calendar-modal` 模态框
- 包含月份控制器、日历网格、图例说明和详情区域

### CSS 样式
- 渐变背景区分假期和补班
- Neumorphism 风格保持一致
- 响应式设计，适配不同屏幕
- 悬停和选中状态的视觉反馈

### JavaScript 功能
- `showHolidayCalendar()`: 显示日历弹窗
- `closeHolidayCalendar()`: 关闭日历弹窗
- `changeHolidayMonth(delta)`: 切换月份
- `renderHolidayCalendar()`: 渲染日历
- `isHolidayDate(date)`: 检查是否是假期
- `isMakeupDate(date)`: 检查是否是补班日
- `showHolidayDetail()`: 显示日期详情

## 注意事项
- 所有日期均基于2026年法定节假日安排
- 补班日期根据国务院发布的调休安排设置
- 如需更新数据，请修改 `script.js` 中的 `holidays` 和 `makeupDays` 数组
- 当前实现仅本地更新，未推送到 git（按用户要求）
