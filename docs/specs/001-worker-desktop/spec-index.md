## 1. 功能摘要

### 核心功能概览

"打工人工作桌面"是一个集成了健康打卡、任务管理、节假日提醒、外语学习及情绪记录的轻量化 Web 网页应用。该工具旨在通过便捷的浏览器访问，帮助职场人士平衡工作与生活，提升工作效率的同时关注身心健康。核心模块包括："吾日三省吾身"（健康习惯打卡，采用简单计数制）、"待办清单"（任务管理与统计）、"放假倒计时"（基于内置节假日数据）、"今日单词"（基于内置词库的每日一词）以及"心情日记"（情绪记录与回顾）。

### 关键业务流程

1.  **每日健康打卡**：用户查看"吾日三省吾身"模块 → 点击增加喝水/如厕/运动次数 → 系统简单记录累计次数。
2.  **待办任务管理**：用户新增待办事项 → 勾选完成 → 系统自动将已完成项置灰并沉底 → 实时更新当日/本周的完成率统计。
3.  **单词学习与复习**：系统每日从本地词库随机展示一个单词 → 用户查看今日单词 → 点击小喇叭听发音（TTS） → 加入生词库 → 进入生词库模式进行默写复习。
4.  **心情记录**：用户打开心情日记 → 选择当日心情并输入备注 → 保存 → 在日历或时间轴视图中回顾心情趋势。

### 关键数据实体（快速参考）

| 实体名称 | 业务含义 | 关键属性（业务层面） |
| :--- | :--- | :--- |
| **健康打卡** | 用户的每日健康习惯记录 | 打卡类型（喝水/如厕/运动）、日期、状态 |
| **待办事项** | 用户的工作或生活任务 | 内容、状态（未完成/已完成）、创建时间、完成时间 |
| **节假日** | 国家法定节假日信息 | 节日名称、日期、放假安排 |
| **单词** | 每日推荐或收藏的英语单词 | 英文拼写、中文释义、发音、生词标记、掌握程度 |
| **心情日记** | 用户每日的情绪记录 | 心情类型、文本内容、日期 |

### 关键交互点（快速参考）

*   **待办列表交互**：勾选复选框即时改变任务状态并触发列表重排序（完成项沉底）。
*   **单词发音**：点击小喇叭图标播放单词发音。
*   **心情视图切换**：通过切换按钮在"时间轴"和"日历"两种视图间无缝切换心情记录展示。

## 3. 成功标准（全局）

*   **用户活跃度**：用户每日打开应用并完成至少一项（打卡/待办/背词/记心情）操作。
*   **任务完成率提升**：用户通过使用待办统计功能，能够感知并提升每日任务的完成率。
*   **单词记忆效果**：用户在生词库默写功能的正确率随时间推移逐步提升。

## 4. 关键实体（全局定义）

### 4.1 健康打卡记录 (HealthCheckIn)
*   **类型 (Type)**：枚举值（喝水, 如厕, 运动）。
*   **时间 (Timestamp)**：打卡的具体时间。
*   **日期 (Date)**：归属的日期（用于按天统计）。

### 4.2 待办事项 (TodoItem)
*   **内容 (Content)**：任务的具体描述文本。
*   **状态 (Status)**：枚举值（未完成, 已完成）。
*   **创建时间 (CreatedAt)**：任务创建时间。
*   **完成时间 (CompletedAt)**：任务被标记完成的时间（用于统计）。
*   **归属日期 (Date)**：任务所属的日期（用于"当日"统计）。

### 4.3 节假日 (Holiday)
*   **名称 (Name)**：节日名称（如：国庆节）。
*   **开始日期 (StartDate)**：放假开始日期。
*   **天数 (Duration)**：放假天数。

### 4.4 单词 (Vocabulary)
*   **拼写 (Spelling)**：英文单词拼写。
*   **释义 (Meaning)**：中文解释。
*   **音频源 (AudioUrl)**：发音文件链接或TTS标识。
*   **生词标记 (IsNewWord)**：布尔值，是否被加入生词库。
*   **掌握程度 (MasteryLevel)**：用于复习功能的熟练度标记。

### 4.5 心情日记 (MoodEntry)
*   **心情类型 (MoodType)**：枚举或图标ID（如：开心, 一般, 难过）。
*   **备注 (Note)**：用户输入的文本内容。
*   **日期 (Date)**：记录日期。

## 5. 假设与依赖（全局）

*   **假设**：用户浏览器支持现代 Web 标准（如 TTS API）。
*   **假设**：用户无需跨设备同步数据，数据仅存储在本地（LocalStorage/IndexedDB）。
*   **依赖**：无外部 API 依赖，所有功能离线可用。
