# US-3: 节假日盼头

**优先级**：Medium
**父文档**：[spec-index.md](../spec-index.md)

## 1. 用户场景和测试

**作为** 一名期待假期的员工，
**我想要** 知道距离下一个法定节假日还有多少天，
**以便于** 调整工作心态，安排假期计划。

### 验收场景

*   **验收场景 3.1：查看倒计时**
    *   **操作**：用户查看"放假倒计时"模块。
    *   **结果**：显示文案"距离 [下一个法定节假日名称] 假还有 [X] 天"。

---

## 2. 功能需求

### 模块三：放假倒计时

*   **FR-301**：系统应内置当前年度的国家法定节假日数据（JSON格式）。
*   **FR-302**：系统应自动计算当前日期距离下一个法定节假日的天数。
*   **FR-303**：系统应展示固定格式文案："距离 [节日名称] 假还有 [天数] 天"。

---

## 3. 相关实体

*   **节假日 (Holiday)**
    *   **名称 (Name)**：节日名称（如：国庆节）。
    *   **开始日期 (StartDate)**：放假开始日期。
    *   **天数 (Duration)**：放假天数。
