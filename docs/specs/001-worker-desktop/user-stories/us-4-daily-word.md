# US-4: 英语碎片学习

**优先级**：Medium
**父文档**：[spec-index.md](../spec-index.md)

## 1. 用户场景和测试

**作为** 一名希望利用碎片时间提升自我的员工，
**我想要** 每天学习一个新单词并能复习生词，
**以便于** 积累英语词汇量。

### 验收场景

*   **验收场景 4.1：学习今日单词**
    *   **操作**：用户点击单词旁边的"小喇叭"图标。
    *   **结果**：系统播放该单词的标准发音。
*   **验收场景 4.2：加入生词库**
    *   **操作**：用户点击"加入生词库"按钮。
    *   **结果**：该单词被标记为生词，可在生词库中查看。
*   **验收场景 4.3：默写复习**
    *   **操作**：用户在生词库中启动"默写模式"。
    *   **结果**：系统隐藏单词拼写，仅显示中文释义，供用户输入拼写进行验证。

---

## 2. 功能需求

### 模块四：今日单词

*   **FR-401**：系统内置英语高频词库（如 CET-4/6 或职场常用词），每日随机展示一个。
*   **FR-402**：**点读功能**：调用浏览器原生 TTS (Text-to-Speech) 接口朗读单词。
*   **FR-403**：**生词库管理**：用户可将今日单词一键加入"生词库"。
*   **FR-404**：**复习默写**：系统应提供生词复习功能，支持"默写模式"（隐藏英文，看中文写英文，或听音写词）。

## 3. 相关实体

*   **单词 (Vocabulary)**
    *   **拼写 (Spelling)**：英文单词拼写。
    *   **释义 (Meaning)**：中文解释。
    *   **音频源 (AudioUrl)**：发音文件链接或TTS标识。
    *   **生词标记 (IsNewWord)**：布尔值，是否被加入生词库。
    *   **掌握程度 (MasteryLevel)**：用于复习功能的熟练度标记。
