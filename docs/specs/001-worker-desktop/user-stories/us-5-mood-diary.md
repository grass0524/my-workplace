# US-5: 情绪记录与回顾

**优先级**：Medium
**父文档**：[spec-index.md](../spec-index.md)

## 1. 用户场景和测试

**作为** 一名注重心理健康的用户，
**我想要** 记录每天的心情并能按时间回顾，
**以便于** 觉察自己的情绪变化趋势。

### 验收场景

*   **验收场景 5.1：记录心情**
    *   **操作**：用户选择今日心情（如：开心、疲惫）并输入"今天项目上线顺利"。
    *   **结果**：心情日记保存成功。
*   **验收场景 5.2：切换视图回顾**
    *   **操作**：用户点击"日历视图"按钮。
    *   **结果**：心情日记列表切换为日历样式，日历格子上显示对应日期的心情图标。

---

## 2. 功能需求

### 模块五：心情日记

*   **FR-501**：用户可录入当日心情，包括选择心情类型（如表情图标）和输入文本备注。
*   **FR-502**：**统计功能**：系统应统计不同心情类型的出现频率（如：开心 5天，焦虑 2天）。
*   **FR-503**：**多视图展示**：系统应提供两种展示视图：
    *   **时间轴视图**：按时间倒序列表展示心情日记。
    *   **日历视图**：在月历上以图标或颜色标记每日心情。

---

## 3. 相关实体

*   **心情日记 (MoodEntry)**
    *   **心情类型 (MoodType)**：枚举或图标ID（如：开心, 一般, 难过）。
    *   **备注 (Note)**：用户输入的文本内容。
    *   **日期 (Date)**：记录日期。
