# 设计工作流程指南

## 从 Pencil 到设计软件的工作流程

### 步骤 1：导出设计截图
使用 Pencil 生成 PNG 截图，然后导入到以下软件：

### 支持的设计软件：

#### 1. **Figma**（推荐）
- 导入 PNG 作为参考图
- 使用 Figma 重新绘制设计
- 优点：协作、组件系统、自动布局
- 网址：https://figma.com

#### 2. **Sketch**（Mac only）
- 导入 PNG 作为参考
- 重新绘制设计
- 优点：符号系统、插件丰富

#### 3. **Adobe XD**
- 导入 PNG 作为参考
- 重新绘制设计
- 优点：与其他 Adobe 软件集成

#### 4. **Photoshop/Illustrator**
- 导入 PNG 进行进一步编辑
- 添加视觉效果、标注

## 推荐方案：直接使用 HTML/CSS

### 为什么推荐 HTML/CSS 作为设计工具？

1. **实时预览** - 保存后刷新即可看到效果
2. **精确控制** - CSS 可以实现新拟物的所有效果
3. **无需转换** - 设计即代码，代码即设计
4. **浏览器开发工具** - 最强大的设计调试工具
5. **响应式** - 自动适配不同屏幕尺寸

### 工具推荐：

| 工具 | 用途 |
|------|------|
| Chrome DevTools | 实时编辑样式 |
| VSCode + Live Server | 自动刷新预览 |
| CSS Grid/Flexbox | 布局调试 |
| CSS Variables | 全局样式管理 |

---

## 快速开始：在浏览器中设计

1. 在浏览器中打开 `login.html`
2. 按 `F12` 打开开发者工具
3. 点击"选择元素"工具（或按 `Ctrl+Shift+C`）
4. 点击任何元素进行编辑
5. 在 Styles 面板中实时调整样式
6. 满意后复制 CSS 到代码文件

---

## 结论

**对于新拟物风格设计，推荐直接使用 HTML/CSS + 浏览器开发工具**

原因：
- 新拟物效果依赖 CSS `box-shadow`
- 浏览器能完美渲染这些效果
- 专业设计软件反而难以实现这些 CSS 效果
